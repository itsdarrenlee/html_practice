This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML To Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header include a partition for intro content
p paragraph include a paragraph of text

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication<em> technical sophistication
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a million people</strong> Technical sophistication
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</strong> Technical sophistication
img image tag include an image <img src="http://bit.ly/1MZAFuQ alt"="Michael Hartl">

Div Elements

Tag Name Purpose Example Result
div div Defines a section in a document <div>technical sophistication</div>

Span Elements

Tag Name Purpose Example Result
span span Defines a section in a document <span>technical sophistication</span>

Document Tags

Tag Name Purpose Example Result
html lang=en html defines the html type, as well as it's language <html>technical sophistication</html>
body body defines the body section of the HTML <body>technical sophistication</body>
head head container for metadata, placed between html and body tag <head>technical sophistication</head>
title title defines the title section of the HTML <title>technical sophistication</title>
meta meta describes meta within a HTML document <meta>technical sophistication>
